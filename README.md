# Imango
Image processing library for manual/hybrid content based image recognition, built entirely from scratch using no non standard libraries in C. <br />
Includes:<br />
Canny edge detetection with greyscale conversion, gaussian blurring, sobel operation, non-maximum suppression and hysteresis thresholding.<br />
Shi-tomasi corner detection using gaussian blur and structure tensor matrix <br />
Blob detection using the difference of gaussians
